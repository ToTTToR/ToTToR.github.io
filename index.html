Qu'est ce que Pokemon?

Les Pokémon ont marqué plusieurs générations d'enfants dans les cours de récréation. Entre les jeux vidéo et les cartes à collectionner, Pokémon est devenu un phénomène incontournable. Aujourd'hui encore les enfants sont toujours autant attirés par ces créatures qui se comptent désormais par centaines. De nombreux adultes sont également des joueurs inconditionnels de Pokémon avec les différentes versions du jeu qui apportent leur lot de nouveautés. Si vous souhaitez tout savoir sur les Pokémon, vous ne pouvez pas vous contenter de jouer quelques minutes sur une console Nintendo Switch ou sur Android. Pokémon est devenu tout un univers mondialement connu avec de nombreux produits dérivés. L'origine de ces petites créatures vient pourtant d'une idée toute simple, presque enfantine.
